# Venüs Robotik - Arduino 

Venüs Robotik Medikal Protez A.Ş. bünyesinde gerçekleştirilen Arduino tabanlı 
gömülü sistem uygulamaları.

**Eğitmen:** Ünal Dalkılıç  
**Stajyer:** Muhammed Hamza Karamanlı

## Projeler

### Temel I/O
- Dijital ve analog veri işleme
- LED kontrolü

### PWM ve Joystick
- Analog veri okuma
- PWM ile LED parlaklık kontrolü

### UART Haberleşme
- İki Arduino arası kablolu haberleşme
- Master-Slave mimarisi

### Bluetooth Haberleşme
- HC-05/HC-06 ile kablosuz haberleşme
- Joystick ile uzaktan servo kontrolü
- Klavye ile uzaktan LED kontrolü

## Kullanılan Donanım
- Arduino Uno (x2)
- HC-05, HC-06 Bluetooth modülleri
- SG90 Servo Motor
- Joystick Modülü
- LED, direnç, breadboard

## Kütüphaneler
- `SoftwareSerial.h`
- `Servo.h`
